Fallout Hacker

1. Input

Enter the words from a Fallout hacking puzzle, separated by spaces or commas or any other non-word character.

2. Pick the word at the top of the table

3. Click that word's likeness

4. Repeat

What is this?

Fallout is a registered trademark of Bethesda.

Fallout 3, Fallout New Vegas, and Fallout 4 all use the same "hacking" system to allow the player to gain unauthorized access to computer terminals, based on guessing a password. However, unlike real password guessing, the user is given multiple-choice options and feedback guiding him to the answer. When you guess the wrong word, the game tells you how many of its letters are correct. This number counts only the right letter in the right place, so "abc" and "acb" have a likeness of 1, not 3.

There is an optimal strategy that can solve any of these puzzles.

How does it work?

At each step, the word you chose and the likeness number rule out some of the possible answers, and if you keep track after each guess, eventually only one option will remain.

But you can do slightly better than that. You can choose your guesses in an order that maximizes the speed with which answers are eliminated. This means to pick the word that splits the other words into as many likeness groups as possible, with each group as small as possible.
